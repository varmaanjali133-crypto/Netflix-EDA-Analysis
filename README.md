# Netflix EDA Analysis

## Project Overview

This project explores Netflix's content library using Exploratory Data Analysis (EDA).

The objective was to identify content trends, genre distribution, country contributions, content growth patterns, and Netflix's content acquisition strategy through data visualization and business insights.

---

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Squarify
* Google Colab

---

## Key Analysis Performed

* Missing Value Analysis
* Movies vs TV Shows Analysis
* Ratings Distribution
* Growth of Netflix Content Over Time
* Top Countries Producing Netflix Content
* Top Directors on Netflix
* Movie Duration Analysis
* Content Added Over Time
* Genre Analysis

---

## Key Insights

* Netflix content additions peaked in 2019.
* International Movies and Dramas dominate the platform.
* The United States and India contribute a significant share of content.
* Movies significantly outnumber TV Shows.
* Netflix follows a strong global content strategy.
